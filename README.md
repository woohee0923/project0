# 프로젝트 설명
기본적인 로그인과 회원가입을 통해 각 기능에 접속할 수 있도록 만들었습니다.
사이트 내에는 명언 작성 페이지, 악기 연주 페이지, 해결책 찾기 페이지, 영상 재생 페이지를 만들어 4가지 방법으로 힐링할 수 있도록 구성했습니다.
